# Imaya Laundry - AI-Powered Premium Laundry Services

A modern, responsive landing page for an AI-powered laundry service platform built with Bootstrap 5 and vanilla JavaScript.

## 🌟 Features

✨ **Modern Design**
- Animated gradient backgrounds with floating orbs
- Glass-morphism UI elements
- Smooth scroll animations
- Fully responsive (mobile, tablet, desktop)

🎯 **Sections**
- Hero section with call-to-action
- Services showcase (Wash & Fold, Dry Cleaning, Ironing, Pickup & Delivery)
- Dynamic pricing plans (Basic, Premium, Enterprise)
- Customer testimonials
- Contact information with Google Maps integration
- Newsletter subscription
- WhatsApp floating button for instant booking

⚡ **Performance**
- CDN-based resources (Bootstrap, Font Awesome, Google Fonts)
- No build process required
- Lightweight vanilla JavaScript
- Optimized for all browsers

## 🚀 Quick Start

1. **Open in Browser**: Simply download and open `index.html` in any modern web browser
2. **Deploy Online**: Upload to GitHub Pages, Netlify, or any web server
3. **No Installation Required**: Everything is self-contained in a single HTML file

## 🛠️ Technologies Used

- **HTML5** - Semantic structure
- **CSS3** - Advanced styling with CSS Grid & Flexbox, animations
- **Bootstrap 5.3.2** - Responsive framework
- **Font Awesome 6.5** - Icon library
- **Google Fonts** - Inter & Orbitron typefaces
- **Vanilla JavaScript** - Interactive features (smooth scroll, navbar effects)

## 📝 Customization

### Update Contact Information
Find and replace these values in `index.html`:
- Phone: `+254700000000` → Your phone number
- Email: `info@imayalaundry.com` → Your email
- Address: `123 Laundry Street, Nairobi, Kenya` → Your address
- WhatsApp Link: Update phone number in all WhatsApp links

### Update Pricing
Edit the pricing cards in the "PRICING SECTION":
- Plan names
- Monthly costs
- Features list

### Update Services
Modify the service cards with:
- Service names
- Descriptions
- Pricing

### Update Testimonials
Replace customer testimonials with real reviews:
- Names
- Locations
- Quotes
- Avatars (initials)

## 🔧 Deployment

### GitHub Pages
1. Push files to GitHub repository
2. Go to Settings → Pages
3. Select `main` branch as source
4. Website will be live at `https://Imayake.github.io/imaya_laundry-work`

### Netlify
1. Drag and drop `index.html` to Netlify
2. Or connect GitHub repository
3. Website deploys automatically on push

### Traditional Web Hosting
1. Upload `index.html` to your hosting server
2. Access via your domain name
